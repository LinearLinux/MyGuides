Configure
git config --global user.name "jaschlak"
git config --global user.email "jaschlak@gmail.com"

// Delete old password
git config --global --unset user.password


git clone <url>

push to sites without remoting
git init
git add <filename.txt>
git status
git commit -m "message" <filename.txt>
git push --mirror <url>

Normal workflow
git init
git remote add origin <repository link> -->>https://github.com/jaschlak/MyGuides.git
git add .
git status
git commit -m "<message>"
git push origin master


Force push
git push origin master --force



Remoting
add origin:
git remote add origin <url>


change origin:
git remote set-url origin <https://github.com/jaschlak/Mytest>

check remote:
git remote -v

remove all remote origin:
git remote rm origin

Remove .git in order to reinitialize
rm -rf .git
git init
